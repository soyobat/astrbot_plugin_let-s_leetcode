# LeetCode 刷题插件

## 功能
| 指令               | 功能                 |
| ---------------- | ------------------ |
| `/lcd`           | 获取当日 LeetCode 每日一题 |
| `/lcr`           | 从 HOT 100 随机抽题     |
| `/lcr sql`       | 随机 SQL 热题          |
| `/lcr interview` | 随机面试题              |
| `/lcr 75`        | 随机 LeetCode 75 题集   |
| `/lcr all`       | 从全部题目中随机抽题        |
| `/lcauto`        | 开启/关闭每日早上9点推送每日一题  |

## 说明
- 使用 LeetCode 中国站 (leetcode.cn) 的 GraphQL API
- 支持获取每日一题和特定题集的随机题目
- 支持自动推送每日一题（每天 9:00）
